
# ai4vl video tools

Video conversion tools for ai4vl.

## Install it from PyPI

```bash
pip install video_tools
```

## Usage

```py
from video_tools import BaseClass
from video_tools import base_function

BaseClass().base_method()
base_function()
```

```bash
$ python -m video_tools
#or
$ video_tools
```
